# Getting Started
Install the dependencies and run the project
```
npm install
npm start
```

Head over to https://vitejs.dev/ to learn more about configuring vite

Password Generator Web App
A secure, customizable password generator built from scratch with HTML, CSS, and JavaScript. Generate strong passwords with adjustable length and options to include/exclude numbers and special characters.

🚀 Features
Adjustable Password Length: Set password length between 8–16 characters

Custom Character Sets: Toggle options to include/exclude numbers and symbols

Two Passwords at Once: Generate two password options to choose from

One-Click Copy: Copy any generated password to clipboard instantly

Input Validation: Clear error messages for invalid input or length limits

Responsive UI: Clean, user-friendly interface

Built from Scratch: No external libraries—pure vanilla JavaScript

| Technology        | Purpose                                                     |
| ----------------- | ----------------------------------------------------------- |
| HTML5             | Structure and semantics                                     |
| CSS3              | Styling and layout                                          |
| JavaScript (ES6+) | Password generation logic, DOM manipulation, event handling |
